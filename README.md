# Sentiment Analysis using NLP Pipeline & Machine Learning

## 📌 Project Overview
This project focuses on building a complete end-to-end Sentiment Analysis system using Natural Language Processing (NLP) techniques and Machine Learning models.

The goal is to process raw text data, convert it into meaningful features, and classify sentiments as **positive** or **negative**.

---

## 🎯 Objectives
- Perform text preprocessing using NLP techniques
- Convert text data into numerical features
- Train multiple machine learning models
- Evaluate and compare model performance

---

## 📂 Dataset
- Source: IMDb Movie Reviews Dataset
- Total Samples: 50,000
- Labels: Positive, Negative

---

## ⚙️ NLP Preprocessing Steps
- Lowercasing
- Removal of punctuation
- Removal of stopwords
- Tokenization
- Stemming
- Removal of URLs and special characters

---

## 🔧 Feature Engineering
- Bag of Words (BoW)
- TF-IDF (Term Frequency - Inverse Document Frequency)

---

## 🤖 Machine Learning Models Used
- Logistic Regression
- Naive Bayes
- Decision Tree

---

## 📊 Model Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score

---

## 📈 Results & Insights
- Logistic Regression achieved the best performance.
- Naive Bayes performed well with faster training.
- Decision Tree showed lower accuracy due to overfitting.
- TF-IDF performed better than Bag of Words.

---

## 🔄 Pipeline Flow
Raw Text → Preprocessing → Feature Engineering → Model Training → Evaluation

---

## 💡 Conclusion
TF-IDF combined with Logistic Regression provided the most accurate results. Proper preprocessing significantly improved model performance.
